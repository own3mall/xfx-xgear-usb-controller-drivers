Section 2

Getting Started

¡@

¡@ ¡@     This user manual is to give users a quick start to enjoy the games using this driver. This section will guide you to get start on playing XFX Game Controller. With simple and straight forward guideline, you will be much more enjoying your game.
1. Quick Start ¡@ ¡@

¡@

 ¡@

There are only 4 simple steps to let you to enjoy the game using XFX Game Controller:

        Step 1: Connecting to PC

        Step 2: Install XFX Programmable Driver

        Step 3: Configure your own setting

        Step 4: Start your games
2. Step 1: Connection ¡@ ¡@
¡@ ¡@

1.          Connect your XFX Game Controller to the computer USB port.
3. Step 2: Installation ¡@ ¡@
¡@ ¡@

1.          Insert the XFX driver CD into your CDROM drive.

2.         Follow the instruction to install. (Need to restart your PC)
4. Step 3: Configuration ¡@ ¡@
¡@ ¡@

1.          Double click the XFX driver icon in the system tray.

2.          Custom and save your setting in the XFX driver GUI.

5. Step 4: Start Games ¡@ ¡@
¡@ ¡@

1.         Activate your games and play the games using XFX Game Controller.
