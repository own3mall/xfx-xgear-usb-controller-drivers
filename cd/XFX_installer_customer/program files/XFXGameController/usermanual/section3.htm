Section 3

Installation

¡@

¡@ ¡@     This section will guide you to install the programmable driver of XFX Game Controller. The driver will enable the user configuration that allows users to select their preferred functions.
1. Procedures ¡@ ¡@

¡@

 ¡@

Step 1: Insert the XFX Driver CD in your CDROM. Then the startup page will be loaded automatically. Please choose the right driver with the correct OS. (Note: Please sure that the XFX Game Controller is connected during installation).


Step 2: Click "Next" to next step.


Step 3: Click "Install" to start installation.


Step 4: Driver will be installed automatically.


Step 5: Click "Finish".


¡@

Step 6: A dialog box will be shown to ask you to restart your system. Click "Yes" if you want to restart your system.


Step 7: After restarting the system, you can start enjoying your games.
